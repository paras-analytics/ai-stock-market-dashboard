AI Stock Market Dashboard

Project Overview:

This project is an AI-based Stock Market Dashboard built using Power BI.
The aim is to analyze stock performance, compare multiple companies, and generate simple AI-based insights.
The focus of this project is on applying data analytics concepts and building a functional dashboard.
________________________________________
Dashboard Pages
1. Overview:	Displays overall performance of selected stocks
•	Includes key metrics such as:
o	Price trends
o	Volume
o	High / Low values
•	Helps in understanding general stock movement
________________________________________
2. Comparison Dashboard :	Allows comparison of multiple stocks
•	Metrics used:
o	Price comparison
o	Percentage returns
o	Trend charts
•	Useful for identifying better-performing stocks
________________________________________
3. AI Insights: Provides basic AI-driven insights
•	Highlights patterns such as:
o	Upward / downward trends
o	Sudden spikes or drops
•	Gives a starting point for AI-based analysis in dashboards
